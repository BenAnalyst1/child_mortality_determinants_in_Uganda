# Analysis of the Determinants of Child Mortality in Uganda

## Data
- The data was obtained from the 2022 Uganda Demographic and Health Survey (DHS). First, I requested for this data from Uganda Bureau of Statistics. Second, I obtained the relevant variables using R. Third, I cleaned the dataset in Stata.

## Outcome
- cleaned data

## Ongoing task
- Exploratory data analysis in Stata and R

## Final task
- Logistic regression analysis of child mortality determinants, but a particular empirical quest on how maternal level of education affects child survival

## Repository contents:
- `.qmd` - R quarto script for data extraction phase
- `dofile` - Stata script for data cleaning phase
- `logfile` - Stata data cleaning log output
