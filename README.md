# Analysis of the Determinants of Child Mortality in Uganda

## Data
- The data was obtained from the 2022 Uganda Demographic and Health Survey (DHS). First, I requested for this data from Uganda Bureau of Statistics. Second, I extracted the relevant variables using R. Third, I cleaned the dataset in Stata, and conducted descriptive data analysis using R.

## Outcome
- Cleaned dataset
- Descriptive statistical data analysis

## going task
- Data interpretation

## Repository contents:
- `.qmd` - R quarto script for data extraction phase
- `dofile` - Stata script for data cleaning phase
- `logfile` - Stata data cleaning log output
- `.qmd` - R code for descriptive data analysis
